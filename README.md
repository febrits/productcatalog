# ProductCatalog — E-Commerce Product Catalog

Browse products with cart and checkout flow.

**Live:** https://productcatalog-phi.vercel.app

## Features
- Product grid with filters
- Shopping cart
- Checkout flow
- Product details

## Tech Stack
React 19 · TypeScript · Tailwind CSS v4 · Vite

## Getting Started
```bash
npm install
npm run dev
```
